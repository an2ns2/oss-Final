# 심야의 바텐더

**사용자의 그날 기분(추구미), 선호하는 맛, 원하는 알코올 도수(0~40도)를 조합해 최적의 칵테일을 제안해 주는 맞춤형 추천 웹 서비스입니다.**


---

## 🚀 핵심 특징 및 기술 스택
* **프론트엔드 (Streamlit):** 직관적인 사용자 입력 인터페이스(버튼, 슬라이더)와 감성적인 결과 카드 렌더링.
* **백엔드 (FastAPI):** 가중치 기반 스코어링 및 타이브레이킹(동점자 난수 추출) 알고리즘 적용. 유지보수를 위해 `cocktails.json`으로 지식 베이스(DB) 완전 분리.
* **인프라 (Docker & AWS EC2):** 프론트엔드와 백엔드를 독립적인 컨테이너로 분리(Docker Compose)하여 AWS EC2 환경에 배포 완료.
